当前，UP探测器有如下的几种设计，FTDR, UP.E, UP.W, UP.X, UP.T。

其中，FTDR、UP.E、UP.T 在 Detector 仓库下有对应的版本，其 GitLab 链接如下：
\begin{itemize}
  \item \textbf{FTDR}（基线设计）：对应 Detector 仓库 \texttt{master} 分支中的 compact/components/UP/branch-run5-baseline/ \link{https://gitlab.cern.ch/lhcb/Detector/-/tree/master/compact/components/UP/branch-run5-baseline?ref_type=heads}{[Link]}。

以 UP.E 为例，UP 采用两站（a/b）四层结构：每站内 X 层与 U/V 层间距 $55~\mathrm{mm}$，两站间距 $280~\mathrm{mm}$，整体位于 $Z \approx 2.5 \sim 2.8~\mathrm{m}$（约在 RICH1 下游、磁铁上游），如 Table.~\ref{tab:up-layer-global-z} 所示。其详细设计布局如 Figure.~\ref{fig:up-e-detail} 所示。每一层包含 12 个 Stave，每个 Stave 上排列 $6 \times 8$ （6为正面3个反面3个）个 Module，每个 Module 上集成 $1 \times 8$ 个 Chip，探测器总尺寸约为

\begin{equation}
	1309~\mathrm{mm} \times 1285~\mathrm{mm}
\end{equation}


该设计下，其辐射长度如下所示

芯片设计空间分辨率（\red{目前}）为

\begin{eqnarray}
	\sigma_x &=& 50~\mathrm{\mu m} \\
	\sigma_y &=& 150~\mathrm{\mu m}
\end{eqnarray}

其在 Moore 中的配置如 Listing.~\ref{lst:up-config} 所示。

\begin{lstlisting}[style=python, label=lst:up-config, caption={UP 探测器配置代码（UPConfig）}]
# Moore/Hlt/RecoConf/python/RecoConf/run5_defaults.py L103-127
class UPConfig(_PixelConfig):
    _DERIVED_KEYS = {"Sx", "Sy"}
    def __init__(self):
        self.PitchX = 0.050 * mm
        self.PitchY = 0.150 * mm
        self.LayerKeep = (
            True,
            True,
            True,
            True,
        )
        self.Eff = 0.98

    @property
    def Sx(self):
        return self.PitchX / sqrt(12)

    @property
    def Sy(self):
        return self.PitchY / sqrt(12)

DEFAULTS_UP = UPConfig()
\end{lstlisting}

在当前的设计下，其主要参数为
\begin{eqnarray}
\sigma_{t} &\sim& 50~\mathrm{ps}\\
\sigma_{x} &=& \sigma_{y} \sim 12~\mathrm{\mu m}\\
\mathrm{Pitch} &=& 55~\mathrm{\mu m}
\end{eqnarray}

其对应的配置代码如 Listing.~\ref{lst:tv-config} 所示：
\begin{lstlisting}[style=python, label=lst:tv-config, caption={TV 探测器配置代码（TVConfig）}]
# Moore/Hlt/RecoConf/python/RecoConf/run5_defaults.py
class TVConfig(_PixelConfig):
    def __init__(self):
        self.TimeResolution = 50 * ps
        self.Pitch = 0.055 * mm
        self.SpatialResolution = 0.012 * mm
        self.Use3D = False
        self.Mask = []
        self.Foil = "Cylindrical"
        self.Detailed = True

DEFAULTS_TV = TVConfig()
\end{lstlisting}
